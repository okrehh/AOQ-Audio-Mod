\# Local build dependencies



The DLL files in this directory are required to build AOQ Audio Mod locally.



Required files:



\- 0Harmony.dll

\- Assembly-CSharp.dll

\- BepInEx.dll

\- PhotonUnityNetworking.dll

\- UnityEngine.dll

\- UnityEngine.AudioModule.dll

\- UnityEngine.CoreModule.dll

\- UnityEngine.UnityWebRequestWWWModule.dll

\- UnityEngine.XRModule.dll



Obtain BepInEx and Harmony from a local BepInEx 5 installation.



Obtain the Unity, Photon, and game assemblies from Attack on Quest.



The dependency DLLs are intentionally excluded from this repository.
